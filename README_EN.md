# agent-memory-system-guide

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

An Agent long-term memory guide for Obsidian-friendly workflows.

## What it is

This skill explains how to build a durable memory stack for agents: a compact `MEMORY.md`, daily notes, memory distillation, and Obsidian backups.

## How to use

- Give the skill to an agent that needs long-term memory.
- Start with the three-layer memory structure.
- Keep raw notes in daily files, then distill the stable parts into `MEMORY.md`.
- Use Obsidian as the long-term archive.

## Included files

- `SKILL.md`: skill contract and workflow
- `INSTALL.md`: a copy-paste installation prompt for agents
